# Полное описание проекта "Reactive Velocity" - Портфолио

## Общая информация

**Название проекта:** Reactive Velocity  
**Тип проекта:** Персональное портфолио разработчика  
**Цель:** Демонстрация навыков, проектов и контактная форма для потенциальных клиентов/работодателей

## Технологический стек

### Основные технологии
- **React 19** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - строгая типизация для повышения надежности кода
- **Vite 7** - быстрый сборщик проектов
- **Tailwind CSS 3.4** - утилитарный CSS-фреймворк
- **Framer Motion** - анимации и переходы
- **React Router DOM** - маршрутизация
- **React Hook Form** - управление формами
- **Lucide React** - иконки
- **React i18next** - интернационализация (многоязычность)

### Серверные технологии
- **Vercel Node Functions** - серверные функции для обработки данных
- **React Google ReCAPTCHA** - защита от ботов

## Логика работы приложения

### Компонентная архитектура
- **Компоненты разделены** на common (общие) и sections (секции)
- **Каждая секция** - отдельный компонент с анимацией при скролле
- **Повторно используемые компоненты** (Button, Card, Modal) в папке common

### Управление состоянием
- **Тема** управляется через React Context (ThemeContext)
- **Формы** управляются через React Hook Form
- **Локализация** через react-i18next
- **Маршрутизация** через React Router DOM

### Анимации
- Используется Framer Motion для всех анимаций
- Анимации при появлении (при скролле в область видимости)
- Hover-эффекты для интерактивных элементов
- Плавные переходы между состояниями

## Бэкенд и интеграции

### Серверные функции (Vercel Functions)
#### sendMessage.ts
- **Расположение:** `/api/sendMessage.ts`
- **Функция:** Обработка формы контактов и отправка в Telegram
- **Логика:**
  1. Проверка метода запроса (только POST)
  2. Валидация обязательных полей
  3. Проверка reCAPTCHA через Google API
  4. Проверка валидности Telegram бот токена
  5. Отправка сообщения в Telegram через Bot API
  6. Обработка ошибок и возврат статуса

#### Переменные окружения (серверные)
- `TELEGRAM_BOT_TOKEN` - токен Telegram бота
- `TELEGRAM_CHAT_ID` - ID чата для получения сообщений
- `RECAPTCHA_SECRET_KEY` - секретный ключ reCAPTCHA

#### Переменные окружения (клиентские)
- `VITE_RECAPTCHA_SITE_KEY` - публичный ключ reCAPTCHA

### Интеграция с Telegram
- **Тип интеграции:** Telegram Bot API
- **Функция:** Отправка заполненных форм в Telegram
- **Формат сообщения:** HTML-форматированное сообщение с деталями
- **Безопасность:** reCAPTCHA верификация

### Защита от спама
- **Google reCAPTCHA v2** - проверка, что пользователь не робот
- **Валидация на сервере** - дополнительная проверка данных
- **Серверная обработка** - форма не отправляется напрямую в Telegram
